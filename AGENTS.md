# WASM Target Agents

This repository owns the official AiLang WASM target package.

WASM owns browser/web packaging, static asset runner behavior, WASM host
integration, browser diagnostics, and WASM-specific CI/CD.

It does not own AiLang language semantics, AiVM evaluation semantics, AiVectra
UI semantics, or package semantics.
